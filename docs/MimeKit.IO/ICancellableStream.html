ICancellableStream Interface

An interface allowing for a cancellable stream reading operation.

Syntax

public interface ICancellableStream

Remarks

This interface is meant to extend the functionality of a System.IO.Stream, allowing the MimeKit.MimeParser to have much finer-grained canellability.

When a custom stream implementation also implements this interface, the MimeKit.MimeParser will opt to use this interface instead of the normal System.IO.Stream.Read(Byte[], int, int) API to read data from the stream.

This is really useful when parsing a message or other MIME entity directly from a network-based stream.
