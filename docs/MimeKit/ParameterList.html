ParameterList Class

A list of parameters, as found in the Content-Type and Content-Disposition headers.

Syntax

public class ParameterList : ICollection<Parameter>, IEnumerable<Parameter>, IList<Parameter>

Remarks

Parameters are used by both MimeKit.ContentType and MimeKit.ContentDisposition.
