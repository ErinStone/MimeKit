ContentObject Class

Encapsulates a content stream used by MimeKit.MimePart.

Syntax

public class ContentObject : IContentObject

Remarks

A MimeKit.ContentObject represents the content of a MimeKit.MimePart. The content has both a stream and an encoding (typically ContentEncoding.Default).

Value

Remarks

If the MimeKit.MimePart was parsed from an existing stream, the encoding will be identical to the MimePart.ContentTransferEncoding, otherwise it will typically be ContentEncoding.Default.
