HeaderList Class

A list of MimeKit.Headers.

Syntax

public sealed class HeaderList : ICollection<Header>, IEnumerable<Header>, IList<Header>

Remarks

Represents a list of headers as found in a MimeKit.MimeMessage or MimeKit.MimeEntity.
