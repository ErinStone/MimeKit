ParseException Class

A Parse exception as thrown by the various Parse methods in MimeKit.

Syntax

public class ParseException : FormatException

Remarks

A MimeKit.ParseException can be thrown by any of the Parse() methods in MimeKit. Each exception instance will have a ParseException.TokenIndex which marks the byte offset of the token that failed to parse as well as a ParseException.ErrorIndex which marks the byte offset where the error occurred.
