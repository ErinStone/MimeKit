CmsRecipient Class

An S/MIME recipient.

Syntax

public class CmsRecipient

Remarks

If the X.509 certificates are known for each of the recipients, you may wish to use a MimeKit.Cryptography.CmsRecipient as opposed to having the MimeKit.Cryptography.CryptographyContext do its own certificate lookups for each MimeKit.MailboxAddress.

Exceptions

Type Reason
ArgumentNullException certificate is null.

Remarks

The initial value of the CmsRecipient.EncryptionAlgorithms property will be set to the Triple-DES encryption algorith, which should be safe to assume for all modern S/MIME v3.x client implementations.

The CmsRecipient.RecipientIdentifierType will be initialized to SubjectIdentifierType.IssuerAndSerialNumber.
