CmsRecipientCollection Class

A collection of MimeKit.Cryptography.CmsRecipient objects.

Syntax

public class CmsRecipientCollection : ICollection<CmsRecipient>, IEnumerable<CmsRecipient>

Remarks

If the X.509 certificates are known for each of the recipients, you may wish to use a MimeKit.Cryptography.CmsRecipientCollection as opposed to using the methods that take a list of MimeKit.MailboxAddress objects.
