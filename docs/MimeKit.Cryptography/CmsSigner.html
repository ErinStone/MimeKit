CmsSigner Class

An S/MIME signer.

Syntax

public class CmsSigner

Remarks

If the X.509 certificate is known for the signer, you may wish to use a MimeKit.Cryptography.CmsSigner as opposed to having the MimeKit.Cryptography.CryptographyContext do its own certificate lookup for the signer's MimeKit.MailboxAddress.
