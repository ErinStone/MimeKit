CryptographyContext Class

An abstract cryptography context.

Syntax

public abstract class CryptographyContext : IDisposable

Remarks

Generally speaking, applications should not use a MimeKit.Cryptography.CryptographyContext directly, but rather via higher level APIs such as MimeKit.Cryptography.MultipartSigned, MimeKit.Cryptography.MultipartEncrypted and MimeKit.Cryptography.ApplicationPkcs7Mime.

Dispose Method

Releases all resources used by the MimeKit.Cryptography.CryptographyContext object.

Syntax

public void Dispose ()

Remarks

Call CryptographyContext.Dispose when you are finished using the MimeKit.Cryptography.CryptographyContext. The CryptographyContext.Dispose method leaves the MimeKit.Cryptography.CryptographyContext in an unusable state. After calling CryptographyContext.Dispose, you must release all references to the MimeKit.Cryptography.CryptographyContext so the garbage collector can reclaim the memory that the MimeKit.Cryptography.CryptographyContext was occupying.
