X509KeyUsageFlags Enum

X.509 key usage flags.

Syntax

[System.Flags]
public enum X509KeyUsageFlags

Remarks

The X.509 Key Usage Flags can be used to determine what operations a certificate can be used for.

Note: a value of X509KeyUsageFlags.None indicates that there are no restrictions on the use of the Org.BouncyCastle.X509.X509Certificate.
