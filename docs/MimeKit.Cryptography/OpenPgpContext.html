OpenPgpContext Class

An abstract OpenPGP cryptography context which can be used for PGP/MIME.

Syntax

public abstract class OpenPgpContext : CryptographyContext

Remarks

Generally speaking, applications should not use a MimeKit.Cryptography.OpenPgpContext directly, but rather via higher level APIs such as MimeKit.Cryptography.MultipartSigned and MimeKit.Cryptography.MultipartEncrypted.
