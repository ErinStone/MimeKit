TnefReader Constructor

Initializes a new instance of the MimeKit.Tnef.TnefReader class.

Syntax

public TnefReader (System.IO.Stream inputStream, int defaultMessageCodepage, TnefComplianceMode complianceMode)

Parameters

inputStream
The input stream.
defaultMessageCodepage
The default message codepage.
complianceMode
The compliance mode.

Exceptions

Type Reason
ArgumentNullException inputStream is null.
ArgumentOutOfRangeException defaultMessageCodepage is not a valid codepage.
NotSupportedException defaultMessageCodepage is not a supported codepage.
MimeKit.Tnef.TnefException The TNEF stream is corrupted or invalid.

Remarks

When reading a TNEF stream using the TnefComplianceMode.Strict mode, a MimeKit.Tnef.TnefException will be thrown immediately at the first sign of invalid or corrupted data.

When reading a TNEF stream using the TnefComplianceMode.Loose mode, however, compliance issues are accumulated in the TnefReader.ComplianceMode property, but exceptions are not raised unless the stream is too corrupted to continue.

Dispose Method

Releases all resource used by the MimeKit.Tnef.TnefReader object.

Syntax

public void Dispose ()

Remarks

Call TnefReader.Dispose when you are finished using the MimeKit.Tnef.TnefReader. The TnefReader.Dispose method leaves the MimeKit.Tnef.TnefReader in an unusable state. After calling TnefReader.Dispose, you must release all references to the MimeKit.Tnef.TnefReader so the garbage collector can reclaim the memory that the MimeKit.Tnef.TnefReader was occupying.

ReadAttributeRawValue Method

Reads the raw attribute value data from the underlying TNEF stream.

Syntax

public int ReadAttributeRawValue (byte[] buffer, int offset, int count)

Parameters

buffer
The buffer to read data into.
offset
The offset into the buffer to start reading data.
count
The number of bytes to read.

Returns

The total number of bytes read into the buffer. This can be less than the number of bytes requested if that many bytes are not available, or zero (0) if the end of the value has been reached.

Exceptions

Type Reason

ArgumentOutOfRangeException

offset is less than zero or greater than the length of buffer.

-or-

The buffer is not large enough to contain count bytes starting at the specified offset.
